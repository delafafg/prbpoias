## Диаграмма декомпозиции первого уровня

```

      │                        │                            │
      ▼                        ▼                            ▼
 ┌──────────┐          ┌──────────────┐           ┌──────────────────┐
 │    A1    │  Сырые   │      A3      │  Чистые   │       A5         │
CSV ──► ИЗВЛЕЧЬ  ├──────► ЗАГРУЗИТЬ   │  данные   │   ВЫПОЛНИТЬ      │
 │  ДАННЫЕ  │  данные  │  В ХРАНИЛИЩЕ ├──────────► │    АНАЛИЗ        │
 └──────────┘          └──────────────┘           └────────┬─────────┘
      │                       ▲                            │
      ▼                       │                   Результаты
 ┌──────────┐        Очищ.    │                   анализа
 │    A2    │  данные │  ┌──────────────┐              │
 │ПРЕОБРАЗО-├─────────┘  │      A4      │              ▼
 │ВАТЬ И    │            │  ПОСТРОИТЬ   │     ┌──────────────────┐
 │ ОЧИСТИТЬ │   ─────────► ВИТРИНУ      │     │       A6         │
```

### Описание блоков

| Блок | Название                    | Вход                    | Выход                    | Управление                      |
|------|-----------------------------|-------------------------|---------------------------|---------------------------------|
| A1   | Извлечь данные              | CSV-файлы               | Сырые DataFrame           | Схема CSV-файлов                |
| A2   | Преобразовать и очистить    | Сырые DataFrame         | Очищенные DataFrame       | Правила валидации, форматы дат  |
| A3   | Загрузить в хранилище       | Очищенные DataFrame     | Таблицы в SQLite (3 шт.)  | Схема БД, ограничения FK        |
| A4   | Построить витрину данных    | Таблицы fact+dim        | Представление sales_mart  | Бизнес-требования к агрегации   |
| A5   | Выполнить анализ            | sales_mart              | Таблицы ABC, XYZ, матрица | Пороги ABC (80/15/5%), XYZ (10/25%) |
| A6   | Сформировать отчёт          | Результаты анализа      | Отчёты, диаграммы PNG     | Форматы отчётов, стандарты      |
